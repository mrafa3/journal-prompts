Journal Prompts API

A free, static JSON API serving 365+ journaling prompts

Click the button to get a prompt

API Endpoint

https://mrafa3.github.io/journal-prompts/journaling-prompts.json

Data Format

{
  "prompts": [
    {
      "prompt": "What made you smile today?",
      "category": "reflection"
    }
  ]
}

Usage Examples

JavaScript

const response = await fetch('https://mrafa3.github.io/journal-prompts/journaling-prompts.json');
const data = await response.json();
const prompt = data.prompts[Math.floor(Math.random() * data.prompts.length)];
console.log(prompt.prompt);

Python

import requests
import random

response = requests.get('https://mrafa3.github.io/journal-prompts/journaling-prompts.json')
prompts = response.json()['prompts']
prompt = random.choice(prompts)
print(prompt['prompt'])

curl

curl -s https://mrafa3.github.io/journal-prompts/journaling-prompts.json | jq '.prompts | .[rand * length | floor]'
